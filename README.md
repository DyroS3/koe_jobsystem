
 
# Koe_JobSystem:
Multijob menu with a built in Boss Menu. 

# Features:
* Have multiple jobs, with the option to clock out(Unemployed)
* Hire, Fire employees
* Add or remove to account funds
* Updates users ranks

# Dependencies:
* [ox_lib](https://github.com/overextended/ox_lib)
